# Feelz Machine

Professional audio sampler with analog-style processing.

## Features
- Real-time audio effects (Tape, Vinyl, Reverb, Delay, Distortion)
- Pitch shifting & time stretching
- Audio visualizer
- Supabase backend
- WAV export

## Setup
1. Create Supabase project
2. Run SQL schema
3. Upload audio files to storage
4. Set environment variables in Netlify
5. Deploy!

## Tech Stack
- React
- Tailwind CSS
- Web Audio API
- Supabase
- Framer Motion
